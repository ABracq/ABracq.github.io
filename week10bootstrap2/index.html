What's Bootstrap?

Bootstrap is like, the best thing ever! You basically put css code in html with a simpler language. With the power of javascript, most of the coding is done for you! You got things like bootstrap nav, which is basically a navigation bar. It's a simple design, but at least you don't have to go through the pain of making a seperate div for the nav bar, add a css page and make it a certain color and do certain things.
